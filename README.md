# Proyecto---Operadores-ineficientes
Este repositorio corresponde al proyecto desarrollado para identificar los operadores ineficientes de una empresa de telefonía ficticia y el impacto que conllevan.

El archivo "operadores ineficientes" corresponde al desarrollo completo del proyecto de análisis del caso de la emrpesa ficticia.

El proyecto fue desarrollado con el entorno "Python 3.13.5" que contaba con todas las librerías necesarias para el desarrollo del proyecto.
